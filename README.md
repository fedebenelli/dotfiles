# dotfiles

Proper documentation someday, these are the dotfiles for my two PCs with one branch corresponding for each one. Since 'lucrecita' is an old netbook that I only use as a server and when I'm on the go I don't recomend much those ones, 'lucrecia' is my main PC.

What I use:
* **distro**: Ubuntu MATE 19.10 x86_64
* **display manager**: lighdm, and login in straight to i3
* **wm**: i3-gaps
* **compositor**: compton-tyrone
* **status**: i3_blocks (I mostly use the example scripts with an added weather one)
* **terminal**: urxvt
* **colorscheme**: to make my colorschemes I use pywal, calling it at login from i3-gaps with a previously established wallpaper
* **text editor**: neo-vim, still setting it up
